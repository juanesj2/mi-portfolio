## 👨‍💻 Sobre Mí

Soy **Juan Stiven**, desarrollador web Junior Full Stack especializado en construir experiencias web rápidas, accesibles y con un diseño impecable. Este portfolio es una muestra de mis habilidades técnicas y mi pasión por el diseño UI/UX moderno.

## 🚀 Tecnologías

Este proyecto está construido con un stack moderno enfocado en el rendimiento y las animaciones fluidas:

- **Frontend Core:** React 19 + Vite.
- **Estilos:** Tailwind CSS v4 con una paleta de colores personalizada (Cyberpunk Elegante / Indigo).
- **Animaciones:** Framer Motion y GSAP para transiciones fluidas de scroll y componentes.
- **Efectos 3D / WebGL:** OGL para fondos envolventes (Aurora) y tarjetas interactivas holográficas.

## 🛠️ Instalación Local

Si quieres clonar este proyecto y probarlo en tu máquina local:

```bash
# 1. Clonar el repositorio
git clone https://github.com/juanesj2/mi-portfolio.git

# 2. Entrar al directorio
cd mi-portfolio

# 3. Instalar las dependencias
npm install

# 4. Iniciar el servidor de desarrollo
npm run dev
```
